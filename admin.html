Quick Tips

How this dashboard works
Edit & Save
Make changes in any section. Click Save Changes at the top. Data is stored in browser memory and exported as a data file.
Images
Upload product photos, hero image, and your portrait from each section. Images are converted and stored locally.
Sync to Website
After saving, click Export Data in Settings → copy the JSON → paste into your website's data file. See the How-To-Sync guide.
